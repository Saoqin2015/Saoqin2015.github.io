当前的打算

Veröffentlicht am 2017-08-27 | in 计划

前言

记录自己不断变化更新的目标。最近越来越健忘，担心自己忘了目标，丢了初心。

具体目标

  1. 深入学习 Spring Boot；

    能够快速应用，熟悉其内在结构、设计思路、应用场景以及配套组合。

  2. 深入学习 Kafka；

    能够快速应用，全面了解其生态形式，同类产品的利弊特点，内部构造，设计思路。

  3. 深入学习 Java 虚拟机；

    能够在解决Java性能问题上能够得到更多的启示，能够对Java应用的开发得到更多的指导。

  4. 深入学习Redis；

    阅读其源码，要搞明白这东西为什么这么受欢迎。

  5. 初步学习React、ES6、NodeJs

    JavaScript将越来越成为这个世界不可或缺的一部分，要跟上节奏啊。通过Hexo的使用，觉得自己改好好反省一下。

  6. 记录自己的学习过程；

    督促自己、分享他人。

  7. 记录自己的Demo项目过程；

    学以致用、实战中摸爬滚打、精炼自己的所学。

  8. 尝试使用go。

    go lang 是未来，自己的Demo项目能不能用go重构呢？非常好奇！

结语

以上目标有具体要求描述，但没有单独设立时间，因为在我看来他们相互之间有很多联系，同步学习是有必要的。决定以项目目标为主线，层层学习，所以对它们定义一个整体的时间表：希望自己在半年内能够对它们融会贯通，有一个较为全面的认识。然后每一项再单独发现难点各个击破。

有时间表的目标才能算有效的目标，希望自己能够说到做到！加油！

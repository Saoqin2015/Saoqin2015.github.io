Hexo[1]-如何在Mac上安装

Veröffentlicht am 2017-08-25 | in Hexo

前言

感谢Hexo、Markdown、github、gitment等工具、平台、插件的发明人。

想开个人技术博客很久了，可是因为自己的拖延症却一直不能如愿。幸运的是终于下决心开博客的是否发现了Hexo、Markdown与githubpage这样的好东东。刚好符合自己目前的需求——不需要数据库和申请虚拟空间就可以发布静态页面内容的博客系统。其次，作为技术人员除了CSDN以外，github是与世界接轨的桥梁，在这上面开个博客有助于和各路技术人员交流:happy: 。

写该Hexo系列的博客内容主要初衷就是记录自己的使用过程，一方面方便别人，一方面如果自己哪天要重新部署博客系统的时候好有个资料参照。

废话不多说，操练起来。

初来乍到，说的有什么不对的，还请各位看官多指正。

3. 将要使用的工具

  • Ruby

    来自日本的一种脚本编程语言，

  • homeBrew

    类似于ubuntu的apg-get、centOS的yum、open suse的zypper，有统一的软件源，可以用来下载、安装、卸载很多软件包，例如：NodeJS、wget、JDK、golang等等。

    homeBrew官网：https://brew.sh

  • NodeJs

    让JavaScript独立运行的一个开发运行环境，得益于JavaScript的广泛使用、V8引擎的问世和想要把后台一起做了的前端工程师们，于是诞生了这个伟大的作品。Hexo就是运行中NodeJs环境中的。

    NodeJs官网：https://nodejs.org/en/

  • Hexo

    够极速、够简单的博客框架。非常适合个人博客的写作和发布。可是这么好的工具，居然没有百度百科词条？？！支持Markdown的文件格式，能够生成整个博客的静态页面。

    Hexo官网：https://hexo.io/docs/

    摘自官网：Hexo is a fast, simple and powerful blog framework. You write posts in Markdown (or other languages) and Hexo generates static files with a beautiful theme in seconds.

  • Git

    区别于集中式的分布式版本管理工具。开源社区的神器。

    Git官网：https://git-scm.com
